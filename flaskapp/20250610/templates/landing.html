Lexicogrammatical Tagger (LxGrTgr) BETA VERSION

To explore the output provided by LxGrTgr, type (or copy/paste) text into the text box and press the "Run LxGrTgr" button. For more information about the tagger, please visit the LxGrTgr Github page.
Please note: This sample web application currently uses a reasonably accurate (and particularly fast) NLP model from Spacy (en_core_web_sm). For more accurate results, Spacy's slower and more resource instensive but more accurate en_core_web_trf model should be used.
